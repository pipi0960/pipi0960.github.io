写一篇自己的博客：

恭喜你以将军百战死的心态完成可了博客的创建，当然你的收获的成就感还有快乐也是满满的一大筐，现在我们将这种快乐写成一篇文章，上传到我们的博客，来跟大家分享（得瑟）一下。

下载Typora

于我而言是不可能为了上传博客而学习markdown等一些网站语言。于是Typora像个救世主一样出现在我的世界。虽然他是收费的，但我们有学习版。

  • 这里就下载学习版typora就好了，我们获取到Typora之后就可以写一篇博客了

写博客

  • 我们在myBlog文件夹下右键选择”open git bash here”

  • 我们输入hexo new “text”，然后回车

    然后我们会看到有一个.md的文件生成到了 D:\library\myBlog\source_posts目录下。

  • 我们进入该目录就会发现多了一个text文件

  • 我们用Typora打开它，会发现程序已经帮我们创建好一些参数了。

  • 我们按键盘的Ctrl + 1是创建一级标题，Ctrl + 2是二级标题。

    为了让你的文章看起来更加美观，还有更多内容跟玩法可以慢慢探索

    可以参考Typora官网的教程

  • 写好内容之后我们保存并退出，回到博客的根目录下右键打开git bash，依次输入以下指令,然后过段时间我们就能在网站看到我们新写的文章啦。
