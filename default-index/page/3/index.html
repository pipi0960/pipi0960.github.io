在typora写的博客不能将照片同步到github

发表于 2023-09-14 更新于 2023-11-19

问题：

在自己博客搭建好以后会发现在Typora中插入的图片在公网链接不能查看

为了找到问题我们在浏览器按f12，可以看到图片的地址仍然是本地电脑上的图片路径，那么问题很明显已经找到了，就是Typora并没有将图片上传到GitHub服务器。

解决：

  • 我先尝试了新建一个GitHub仓库然后将这张图片上传到这个仓库。

  • 然后我们将这里的链接复制

  • 在Typora中右键空白处选择插入图像，可以看到软件已经帮我们自动在小括号里填好了该图片的网址，如果没有，手动添加一下就好。

  • 最后我们在git控制台依次运行一下hexo clean，hexo g，hexo d进入我们的网站我们就可以发现图片已经能正常在网页进行浏览了。

​

  • 如果图片还是出不来就挂加速器，然后刷新一下就好了，github的服务器毕竟在国外，还是先忍受以下好了，后期研究以下自己的服务器。
